Conceptos basicos

Aqui veremos los conceptos basicos de Python

Aqui veremos como imprimir informacion en la consola, como almacenar datos en las variables, y las condicionales.

Imprimir informacion en la consola

Para imprimir informacion en la consola, se utiliza la funcion print(). Por ejemplo: 

print("Hola mundo")

Esto imprimira "Hola mundo" en la consola.

Variables

Las variables son cajas que almacenan datos y usar esa informacion luego.

Para crear una variable, simplemente se le asigna un nombre y se le da un valor. Por ejemplo: 

nombre = "Juan"

En este caso, hemos creado una variable llamada nombre y le hemos asignado el valor "Juan".

Condicionales

Las condicionales son estructuras que permiten ejecutar un bloque de codigo si se cumple una condicion. En Python, se utilizan las palabras clave if, elif y else, osea, si pasa esto, haz esto. Por ejemplo: 

if edad >= 18:
  print("Eres mayor de edad")
elif edad > 12:
  print("Eres un adolescente")
